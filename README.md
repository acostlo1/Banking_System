💳 Java Banking System (Console Application)

A simple banking system built in Java that allows users to manage their account through a console-based interface. This project demonstrates core programming concepts such as methods, loops, conditionals, and user input handling.

🚀 Features
View current account balance
Deposit funds with input validation
Withdraw funds with insufficient balance protection
Interactive menu system using loops and switch statements
🧠 Concepts Used
Methods (void and return types)
Loops (while)
Conditional statements (if, switch)
User input with Scanner
Basic program structure and state management
📌 Notes

This project was created as part of my Java learning journey to reinforce foundational concepts and improve problem-solving skills.
